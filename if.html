If...Else Statements

  • if...else: two choices. ie Like checking if you can enter a club: if age ≥ 18, let in; else, deny.
  • if...else if...else: many choices. ie Like checking exam scores: if score ≥ 90, grade A; else if score ≥ 80, grade B; else, grade C.
  • for loop: fixed repetitions. ie Like counting from 1 to 5.
  • while loop: repeat until false. ie Like waiting for user input.
  • do...while loop: run once, then check. ie Like showing a menu and then checking if the user wants to continue.
