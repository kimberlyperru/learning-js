Operations

== (Equal to) they are the same value even though they look different. ie 5 == "5" true

!= (Not equal to) they are not the same value. ie 5 != "5" false

=== (Strict equal to) they are the same value and type. ie 5 === 5 true

!== (Strict not equal to) they are not the same value or type. ie 5 !== "5" true

> (Greater than) they are not the same value. ie 5 > 4 true

< (Less than) they are not the same value. ie 5 < 6 true

>= (Greater than or equal to) they are not the same value. ie 5 >= 5 true

<= (Less than or equal to) they are not the same value. ie 5 <= 6 true
